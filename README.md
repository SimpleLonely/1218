# 金融深度学习系统的测试策略设计与错误实证分析



### 具体任务目标

- 初步设想利用 TensorFlow 框架搭建对现货黄金价格进行预测的深度学习模型

- 进行分钟回测后，定位出模型失效状况出现的区间

- 分析其可能原因和规律，针对性地提出应对的策略。

预期成果为实现一个基于历史变动和影响因子的现货黄金价格预测的深度学习模型，以及一份金融领域深度学习范式的完善方案。



### 任务分工

##### 算法组

> 训练神经网络
>
> 模型比较择优
>
> 对照回测定位
>
> 错误分析总结

##### 数据组

> 文献阅读确定模型
>
> 获取&爬取参数
>
> 有效性分析
